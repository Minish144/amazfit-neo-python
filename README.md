# amazfit-neo-python

## Enable bluetooth
```bash
$ bluetoothctl

[bluetooth] power on
```

## Installing dependencies
```bash
$ python3 -m venv env

$ source env/bin/activate

$ pip install -r requirements.txt
```

## Setting MAC address and auth key
set `MAC_AMAZFIT_NEO` and `KEY` in main.py

## Running
```bash
python3 main.py
```
